# NebulaOS

NebulaOS — экспериментальная модульная операционная система для нестабильных сред и разнородного железа. Интерфейс минималистичен, с упором на текст, встроенный терминал и быстрый старт для секретных или офлайн‑терминалов.

## Возможности

- Модульная архитектура ядра и сервисов  
- Быстрая загрузка с поднятием только критических компонентов  
- Минималистичный текстовый интерфейс с псевдографикой  
- Встроенный терминал для администрирования и разработки  
- Режим работы без сети для повышения безопасности

## Статус проекта

Проект находится в активной разработке. Интерфейсы и внутренние API могут меняться.
